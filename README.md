# 🎮 Copilot Combate Game

Game de combate para treinamento Microsoft 365 Copilot - Derrote os vilões da produtividade!

## 🚀 Quick Start

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build
```

## 🎯 Vilões

- **🐝 Round 1:** Mestre das Notificações
- **🦍 Round 2:** Capitã Pesquisa Infinita  
- **👻 Round 3:** Senhora Perfeccionista
- **🤖 Round 4:** ControlC+V

## 🛠️ Stack

- **Frontend:** React 18 + TypeScript + Vite + Tailwind CSS
- **Backend:** Firebase (Auth, Firestore, Functions)
- **Hosting:** Vercel

## 📦 Estrutura

```
packages/
├── frontend/     # React App
├── functions/    # Firebase Cloud Functions (futuro)
└── shared/       # Tipos compartilhados (futuro)
```
